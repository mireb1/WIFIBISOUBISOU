Questions Fréquentes

Comment utiliser mon voucher WiFi ?

1. Connectez-vous au réseau "WiFi-Bisou-Bisou"

2. Ouvrez votre navigateur

3. Entrez votre code voucher dans la page de connexion

4. Cliquez sur "Se connecter"

5. Profitez de votre connexion!

Combien de temps mon voucher est-il valide ?

• Forfait Basique : 24h pour activer, 1h de connexion

• Forfait Premium : 48h pour activer, 6h de connexion

• Forfait Journalier : 72h pour activer, 24h de connexion

Que faire si mon voucher ne fonctionne pas ?

1. Vérifiez que vous êtes connecté au bon réseau

2. Assurez-vous que votre voucher n'a pas expiré

3. Contactez notre support si le problème persiste

4. Gardez votre reçu de paiement pour référence

Comment obtenir un remboursement ?

Les remboursements sont possibles dans les 24h suivant l'achat pour les vouchers non utilisés. Contactez notre support avec votre code de transaction Flexpaie.
